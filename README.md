# 🚀 Overview

Cornerstone Publications is a full-stack web platform developed to streamline the complete academic publishing workflow. It enables researchers, authors, reviewers, editors, and administrators to manage journals efficiently through an intuitive and secure interface.

The platform integrates AI-powered manuscript analysis, secure authentication, automated workflows, online payment processing, and role-based dashboards to modernize academic publishing.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/dev-son123/Cornerstone-Publications.git
```

Move into the project

```bash
cd Cornerstone-Publications
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

---

# 🌐 Environment Variables

Create a `.env` file and configure:

```env
VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_KEY
```
